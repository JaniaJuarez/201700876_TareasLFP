Diferencia entre los Lenguajes de Programacion Imperativos y Declarativos

Que el imperativo se enfoca en el "COMO" más que todo es el lenguaje que 
le indica a la computadora como realizar una instrucción.
Mientras que el declarativo se enfoca en "QUE" nos dice que es lo que se 
tiene que realizar y una pequeña ventaja es que a veces se puede realizar con 
solamente una línea de código.

Ejemplo Imperativo

$listaparticipantes = [1 => 'Peter', 2 => 'Hans', 3 => 'Sarah'];
$nombres = [];
foreach ($listaparticipantes as $id => $apellido) {
    $nombres[] = $apellido;
}
Ejemplo Declarativo
Se puede observar que es el mismo que el imperativo y realiza lo mismo pero en una línea.

$nombres = array_values($listaparticipantes);
